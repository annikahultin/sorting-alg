Compilations Instructions:
javac Sort.java
javac DataGenerator.java

Running Instructions:
java Sort -i/-m/-q 10 34 12 3 45
-i = sort with Insertion Sort
-m = sort with Merge Sort
-q = sort with Quick Sort
** add all numbers in the sequence you want to be sorted after you specify the sorting alg **
java DataGenerator i/m/q
i = average of 10 runs on input sizes 1,000-100,000 on Insertion Sort
m = average of 10 runs on input sizes 1,000-100,000 on Merge Sort
q = average of 10 runs on input sizes 1,000-100,000 on Quick Sort
